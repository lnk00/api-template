## Context

This repositrory is a template for all the API i'm shipping. A basic graphql resolver is setup aswell as a clasic REST routes handler.

## Setup

- ### run in dev mode

  - `pnpm i`
  - `pnpm dev`

- ### build

  - `pnpm clean`
  - `pnpm build`

- ### lint

  - `pnpm run lint`
